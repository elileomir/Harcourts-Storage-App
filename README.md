# Harcourts Storage App

> **A modern, intelligent solution for managing storage facilities.**

Welcome to the Harcourts Storage App. This platform is designed to streamline the management of storage units, bookings, and customer interactions, providing a seamless experience for both administrators and users.

---

## 🌟 Overview

The Harcourts Storage App is a comprehensive tool built to simplify the complexities of storage facility management. It combines intuitive design with powerful functionality to help you:

*   **Manage Units Effortlessly**: Keep track of unit availability, sizes, and status in real-time.
*   **Streamline Bookings**: Handle customer reservations and move-ins with a smooth, digital workflow.
*   **Gain Insights**: Access clear analytics and reports to understand occupancy and performance.
*   **Enhance Communication**: Integrated tools to keep in touch with your customers.

We've built this with a focus on **simplicity**, **speed**, and **reliability**, ensuring that you can focus on your business while the app handles the details.

---

## 🚀 Key Features

*   **Dashboard**: A central hub for a quick overview of your facility's health.
*   **Unit Management**: Detailed inventory control for all your storage units.
*   **Booking System**: Easy-to-use interface for managing customer reservations.
*   **User Administration**: Secure access control for staff and administrators.
*   **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices.

## 🛠️ Developer Guide

This section is for the technical team responsible for maintaining and deploying the application.

### Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (React)
*   **Styling**: Tailwind CSS
*   **Database/Auth**: Supabase
*   **Deployment**: Netlify

### Getting Started

To set up the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/elileomir/Harcourts-Storage-App.git
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the app:**
    Visit [http://localhost:3000](http://localhost:3000) in your browser.

### Building for Production

To create an optimized build for deployment:

```bash
npm run build
```

### Linting

To check for code quality issues:

```bash
npm run lint
```
